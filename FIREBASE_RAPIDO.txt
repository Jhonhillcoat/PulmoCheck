PASO 1: Instalar Firebase CLI
------------------------------
npm install -g firebase-tools


PASO 2: Login y Crear Proyecto
-------------------------------
firebase login

Luego ve a: https://console.firebase.google.com
- Clic en "Add project"
- Nombre: pulmocheck
- Desactiva Google Analytics (opcional)
- Create project


PASO 3: Configurar en tu carpeta
---------------------------------
cd C:\PulmoCheck
firebase init hosting

Responde:
- Use existing project? YES
- Select: pulmocheck
- Public directory: frontend/build
- Configure as single-page app? YES
- Set up automatic builds? NO


PASO 4: Build y Deploy
-----------------------
cd frontend
npm run build
cd ..
firebase deploy --only hosting

¡LISTO! Tu app estará en:
https://pulmocheck.web.app
