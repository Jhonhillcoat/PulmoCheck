PASO 1️⃣  SUBIR CÓDIGO A GITHUB
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. Crear cuenta en: https://github.com
  
  2. En PowerShell (en C:\PulmoCheck):
  
     git init
     git add .
     git commit -m "Initial commit"
     
  3. Crear repo en GitHub → pulmocheck
  
  4. Conectar y subir:
  
     git remote add origin https://github.com/TU-USUARIO/pulmocheck.git
     git branch -M main
     git push -u origin main


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PASO 2️⃣  DESPLEGAR EN RENDER.COM
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. Ir a: https://render.com
  
  2. Sign up con GitHub
  
  3. BACKEND:
     New + → Web Service
     Repo: pulmocheck
     Root: backend
     Build: pip install -r requirements.txt
     Start: gunicorn --bind 0.0.0.0:$PORT app:app
     
     Variables de entorno:
     - FLASK_ENV = production
     
     → Clic "Create Web Service"
     → COPIAR la URL (ej: pulmocheck-api.onrender.com)
     
  4. FRONTEND:
     New + → Static Site
     Repo: pulmocheck
     Root: frontend
     Build: npm install && npm run build
     Publish: build
     
     Variables de entorno:
     - REACT_APP_API_URL = https://TU-BACKEND.onrender.com/api
     
     → Clic "Create Static Site"


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PASO 3️⃣  CONFIGURAR CORS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. Editar backend/app.py
  
     Cambiar:
     CORS(app)
     
     Por:
     CORS(app, origins=["https://TU-FRONTEND.onrender.com"])
     
  2. Guardar y subir:
  
     git add backend/app.py
     git commit -m "Update CORS"
     git push
     
  3. Render redesplegará automáticamente (2-3 min)
